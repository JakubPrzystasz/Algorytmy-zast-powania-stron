# Algorytmy-zast-powania-stron

Trzeba napisać algorytmy:
- FIFO
- Algorytm optymalny
- LRU ( Least Recently Used )
- LFU ( Least Frequently Used )
- MFU ( Most Frequently Used )
- Algorytm drugiej szansy

Użytkownk podaje:
- ciąg odwołań do stron np ( 1, 2, 3, 4, 1, 2, 5, 1, 2, 3, 4, 5 )
- liczbę dostępnych ramek 

Wynik algorytmu:
- odpowiednia tabelka
- ilość błędów braku strony
